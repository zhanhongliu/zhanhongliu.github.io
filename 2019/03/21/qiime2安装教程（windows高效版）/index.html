qiime2安装教程（windows高效版）

0、优点

  • 占用空间大概15G，比官方教程小9G
  • 单系统操作，不需要虚拟机
  • 占用内存更小，运行更流畅
  • 不需要共享文件夹，数据取用更方便
  • 图形化界面，不再担心命令行看不懂

1、前期准备

miniconda安装包并且重命名为1.sh
下载链接：https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda-1.6.2-Linux-x86_64.sh
cmder下载地址：https://cmder.net/
mini版和完整版任选一个即可，建议安装到D盘

2、调试电脑

在控制面板界面，依次操作：程序>程序和功能>启用或关闭windows功能>适用于Linux的windows子系统（打钩）>重启电脑
注意：普通家庭版系统可能需要提前打开开发者模式，百度下就可以，很简单的。

3、安装ubuntu子系统

科普：这里的乌班图子系统并不是真正的linux系统，只是依赖于windows系统的一个极为精简的乌班图系统。虽然不完全，但是足够我们使用qiime2软件。虽然适用于qiime2，但不保证适用于其他软件，但目前可忽略。
打卡“windows store”也就是windows应用商店，搜索ubuntu，下载ubuntu18（TLS）版。

下载完成后，点击启动，自动安装
安装完成后会提示输入新的用户名和密码
（用户名用英文，密码输6个你记得住的简单数字就好，注意在linux系统中输入的密码是不会显示的，但是只要你输入了，它就会存在的。）

关闭安装界面，并打开cmder，输入

bash

如果出现

(bash)**@***$

就说明安装成功了。

（此时cmder不要关闭，如果不小心关闭了，就再打开cmder，并输入bash）

4、安装miniconda

把1.sh放到D盘里（放到根目录，就是直接放在D盘，不要放在D盘的文件夹里）
在cmder中输入如下命令，安装miniconda

cd /mnt/d

bash 1.sh

然后按住enter，进度条一路向下走，直到出现选yes/no时，手动输入yes，按enter，再enter，，等待至安装完成（约3min），最后再选一个填yes，即可完成。

5、更换conda源

ubuntu的默认conda源在国外，如果直接进行后续操作，相关下载速度会非常慢！
在cmder中输入如下命令：

conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

conda config –set show_channel_urls yes

6、更换安装文件夹

在D盘新建一个名叫qiime2的文件夹

再输入命令：

cd /mnt/d/qiime2

7、安装qiime2(输入每条命令并等待完成后再输入下一条)

第一条命令：

wget https://data.qiime2.org/distro/core/qiime2-2019.1-py36-linux-conda.yml

第二条命令：

conda env create -n qiime2-2019.1 –file qiime2-2019.1-py36-linux-conda.yml

第三条命令：

rm qiime2-2019.1-py36-linux-conda.yml

8、待续…

然后你就可以愉快的使用它啦！
（愉快是不可能愉快的了…后面会更新qiime2的图形界面教程，妈妈再也不用担心你看不懂命令行啦~）
（ps:老师的qiime1没有图形化界面哈，只能用命令行，qiime2开始有图形界面了，教程待续~…)
